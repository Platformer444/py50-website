py50 Library

py50 is the library with basic commands

Importing:

import py50


Commands:

1. print Command

The print command helps to print some text, result of a equation or a variable in the output console

Syntax:
print "Text..."
print Equation...
print Variable...


2. var Command

The var command helps to store a string, number or the result of a equation

Syntax:
var "Value..."
var Number...
var Equation...
